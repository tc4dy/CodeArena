# CODE ARENA

## English

**Code Arena** is a cybersecurity toolbox designed for BEGGINER CTF players, red teamers, and security researchers.(For begginers, hobby project.)

It contains **mini codes, exploits, payloads, and utilities** that are:
- Time-saving
- Dependency-aware (or standalone)
- Ready to use in CTF and daily security tasks
- **Open to development** – you can add your own codes

### Categories

- **Web** – SQLi, XSS, LFI, SSRF, shell upload, auth bypass
- **OSINT** – People search, domain recon, social media scraping, email tracking
- **Crypto** – Classical ciphers, hashes, RSA, XOR, padding oracle
- **Pwn** – Buffer overflow, format string, ROP, heap exploitation
- **Reverse** – Keygens, crackmes, anti-debug, unpacking
- **Forensic** – Disk analysis, memory dump, steganography, PCAP carving
- **Cheatsheet** – One-liners, command references, quick lookup tables

Every code is **functional, well-structured, and battle-tested**.

> **Open for contributions** – Add your own mini codes, exploits, or payloads. Pull requests welcome.
