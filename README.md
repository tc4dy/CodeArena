# CODE ARENA

## English

**Code Arena** is a professional cybersecurity toolbox designed for CTF players, red teamers, and security researchers.

It contains **mini codes, exploits, and utilities** that are:
- Time-saving
- Dependency-aware (or standalone)
- Cross-category (Web, OSINT, Crypto, Pwn, Reverse, Forensic, Cheatsheet)

### Categories

- **Web** – SQLi, XSS, LFI, SSRF, shell upload, auth bypass
- **OSINT** – People search, domain recon, social media scraping, email tracking
- **Crypto** – Classical ciphers, hashes, RSA, XOR, padding oracle
- **Pwn** – Buffer overflow, format string, ROP, heap exploitation
- **Reverse** – Keygens, crackmes, anti-debug, unpacking
- **Forensic** – Disk analysis, memory dump, steganography, PCAP carving
- **Cheatsheet** – One-liners, command references, quick lookup tables

Every code is **functional, well-structured, and battle-tested**.

## Türkçe

**Code Arena** – CTF oyuncuları, kırmızı takım üyeleri ve güvenlik araştırmacıları için profesyonel bir araç setidir.

İçerdiği **mini kodlar, exploitler ve yardımcı araçlar**:
- Zaman kazandırır
- Bağımlılıkları bellidir (veya bağımsızdır)
- Kategoriler arası çalışır

### Kategoriler

- **Web** – SQLi, XSS, LFI, SSRF, shell upload, atlama
- **OSINT** – Kişi bulma, domain araştırma, sosyal medya, e-posta takibi
- **Crypto** – Klasik şifreler, hash, RSA, XOR, padding oracle
- **Pwn** – Taşma, format string, ROP, heap
- **Reverse** – Keygen, crackme, anti-debug, paket açma
- **Forensic** – Disk analizi, bellek dökümü, steganografi, PCAP
- **Cheatsheet** – Tek satırlıklar, komut referansları

Her kod **işlevsel, iyi yapılandırılmış ve savaşta test edilmiştir**.
